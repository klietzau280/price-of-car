# What Drives the Price of a Used Car?

**CRISP-DM Analysis | UC Berkeley PCMLAI Practical Application II**

---

## Summary of Findings

Using a dataset of ~426K used car listings from Kaggle, we built regression models (Linear Regression, Ridge, and Lasso) to identify the key drivers of used car prices. Our best model (Ridge Regression) explains approximately **75% of the variation in used car prices** (R² ≈ 0.75).

### Top Price Drivers — Dealership Takeaways

| Factor | Finding | Action |
|---|---|---|
| **Vehicle Age** | Strongest predictor; newer = significantly higher price | Prioritize 0–6 year old inventory |
| **Odometer** | Lower mileage = higher price, nonlinear effect | Focus on vehicles under 80,000 miles |
| **Condition** | Excellent/Like New commands 20–40% premium over Good | Invest in reconditioning before listing |
| **Drive Type** | 4WD commands a consistent premium | Stock 4WD trucks and SUVs |
| **Manufacturer** | Luxury brands 2–4× higher; domestic trucks retain value well | Align brand mix with target margin goals |
| **Fuel Type** | Diesel and electric price higher on average | Consider expanding EV/diesel inventory |
| **Paint Color** | Minimal consistent impact | Do not pay a premium based on color |

### Model Performance

| Model | Test R² | CV R² |
|---|---|---|
| Linear Regression | 0.7539 | 0.7517 |
| **Ridge Regression** | **0.7539** | **0.7517** |
| Lasso Regression | 0.7534 | 0.7512 |

Ridge Regression is recommended as the production model due to its robustness to multicollinearity from one-hot encoded categorical features.

### Notebook Structure (CRISP-DM)

1. **Business Understanding** — Reframe as a regression task; define success criteria
2. **Data Understanding** — Load data, audit missing values, exploratory visualizations
3. **Data Preparation** — Filter outliers, engineer features (`age`, `log_odometer`, `log_price`), encode categoricals
4. **Modeling** — Linear Regression baseline, Ridge with GridSearchCV, Lasso with GridSearchCV (5-fold CV throughout)
5. **Evaluation** — Model comparison, coefficient interpretation, residual diagnostics
6. **Deployment** — Non-technical findings report with actionable dealership recommendations
